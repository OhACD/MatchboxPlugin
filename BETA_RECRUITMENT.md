# Matchbox Beta Testing - We Need You!

**Matchbox** - A recording-proof social deduction game for Minecraft is ready for beta testing!

## What is Matchbox?

Matchbox is a 7-player social deduction game where players work together to identify and eliminate the Spark (impostor) before it's too late. The twist? **Everything is recording-proof** - even if someone streams their screen, you can't tell who the Spark is!

### Key Features
- **Recording-Proof Design**: Identical inventories, paper-based abilities, hologram chat
- **Three Roles**: Spark (impostor), Medic (can save players), Innocent (detective)
- **Three Phases**: Swipe (3 min), Discussion (30 sec), Voting (15 sec)
- **Special Abilities**: Hunter Vision, Healing Touch, Healing Sight
- **Full Game Loop**: Multiple rounds until win condition

## How to Get Started

1. Download the latest beta build
2. Set up a session with `/matchbox start <name>`
3. Add spawn locations with `/matchbox setspawn <name>`
4. Set discussion area with `/matchbox setdiscussion <name>`
5. Have players join with `/matchbox join <name>`
6. Start the game with `/matchbox begin <name>`
