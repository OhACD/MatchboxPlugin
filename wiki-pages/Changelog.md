## Recent Releases

### [0.9.7] - Latest Release
**World-Local Maps, Observability, and Production Hardening**

Major additions:
- World-local map config workflow with `/mb setup` map tooling
- Full session observability via structured logs and statistics API
- Deterministic role assignment and session-scoped custom ability hooks
- Production hardening with CI/coverage verification and release checklist
- Expanded docs and migration guidance for map creators and admins

See the [full changelog](https://github.com/OhACD/MatchboxPlugin/blob/main/CHANGELOG.md) for detailed changes, fixes, and improvements.

### [0.9.4] - Ability System
- Medic secondary ability system
- Delusion ability for Spark (fake infections)
- Steve skin override improvements
- Fixed default seat locations

### [0.9.3] - Dynamic Voting
- Spark secondary ability system (Hunter Vision or Spark Swap)
- Dynamic voting threshold system
- Voting penalty system
- Enhanced tie handling

### [0.9.2] - Configuration & QOL
- Version checking and update notifications
- Pre-discussion elimination notice
- Nickname support throughout UI
- Debug force start command

### [0.9.1] - Config File Support
- Full configuration system with config.yml
- Seat location system
- Damage and block interaction protection
- Welcome message system
- Default configuration for M4tchb0x map
