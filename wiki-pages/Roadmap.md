# Roadmap

This page tracks planned features and improvements for Matchbox.

## Current Focus

Matchbox is in the **post-rapid iteration phase** (v0.9.x). Development priorities:
- Bug fixes and stability improvements
- Polish and user experience refinements
- Production readiness
- Documentation improvements

## Planned for v1.0

- [ ] Final stability and integration testing
- [ ] Complete documentation review
- [ ] Performance optimization
- [ ] Production deployment guide
- [ ] Comprehensive testing across multiple server configurations

## Future Considerations

- Additional secondary abilities for Spark and Medic
- Enhanced spectator features
- Improved skin fallback handling
- Additional configuration options based on community feedback
- Fine-tuning of dynamic voting thresholds based on playtesting data

## Community Requests

Feature requests and suggestions are tracked in [GitHub Issues](https://github.com/OhACD/MatchboxPlugin/issues) with the `enhancement` label.
