
# Getting Started

## Requirements
- Paper 1.21.10+
- Java 21+

## Install
1. Put `Matchbox.jar` and a compatible `ProtocolLib` jar in `plugins/`.
2. Start the server. `plugins/Matchbox/config.yml` will be created on first run.

## Quick usage
- `/matchbox start <name>` — create & start a session
- `/matchbox join <name>` — join a session
- Use `/matchbox setspawn` and `/matchbox setseat <number>` to configure locations in-game.
