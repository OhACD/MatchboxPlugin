A lightweight social deduction minigame plugin for Minecraft (2–20 players).

## Quick install
1. Place `Matchbox.jar` and a compatible `ProtocolLib` jar into your server's `plugins/` folder.
2. Restart the server (Paper 1.21.10+, Java 21+).
3. The plugin will create `plugins/Matchbox/config.yml` on first run — defaults are ready for the M4tchbox map.

---

## Basic usage
- Player commands: `/matchbox join <session>`, `/matchbox leave`, `/matchbox list`
- Admin commands: `/matchbox start`, `/matchbox setspawn`, `/matchbox setseat`, `/matchbox debugstart`, `/matchbox stop`

For the full command list and configuration options, see `docs/Commands.md` and `docs/Configuration.md`.
