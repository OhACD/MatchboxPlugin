# Commands — Quick Reference

## Player
- `/matchbox join <session>` — Join a session
- `/matchbox leave` — Leave current session
- `/matchbox list` — List active sessions

## Admin
- `/matchbox start <session>` — Create and start a session
- `/matchbox stop <session>` — Stop a session
- `/matchbox setspawn` — Save current location as a spawn
- `/matchbox setseat <number>` — Save current location as a discussion seat
- `/matchbox listspawns` — List saved spawns
- `/matchbox listseatspawns` — List saved seats
- `/matchbox clearspawns` / `/matchbox clearseats` — Clear saved locations (requires confirmation)
- `/matchbox debugstart <session>` — Force start for debugging
- `/matchbox debug` — Show debug information

For more details and permission nodes, consult `docs/Configuration.md` and in-game help.
